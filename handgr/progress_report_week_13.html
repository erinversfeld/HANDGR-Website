Progress

The team submitted and subsequently received ethical clearance for their data gathering process. They have to submit a revised application by the end of the week as a formality.

The deadline for the revised project proposal has been shifted to 14 July to accomodate for the exam period.

The team has continued to pilot their data gathering technique. Several issues were run in to and resolved, however, others still remain.

Meeting minutes

Meetings were held on the 28th and the 29th for the following purposes:

  • 28 June: The team met to pilot their data gathering technique again. Issues were discovered with the manner in which the Leap was storing its data. The Leap and Myo were also intermittently disconnecting from the data gathering process. A check-list was devised for the data gathering set up to ensure that all devices are set up correctly and the process is homogenous across participants.
  • 29 June: The team worked remotely on addressing the issues with the Leap and Myo respectively. The Leap's issues were resolved, but the Myo is still proving to be troublesome.

Additional work

The following deadlines have been set for the next set of deliverables:

  • 30 June: The revised ethical clearance application must be submitted.
  • 7 July: The data capture set up must be finalised and participants must be ready to begin.
  • 14 July: The updated project proposal must be submitted on Vula.

Next milestone

Finalising the data capture set up before recruiting participants.
